# Core Inventory

## Overview

Core Inventory is a React + TypeScript + Vite application for inventory operations and tracking.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui

## Local Development

1. Install dependencies:

```sh
npm install
```

2. Start development server:

```sh
npm run dev
```

3. Build for production:

```sh
npm run build
```

4. Preview production build:

```sh
npm run preview
```

## Testing

Run unit tests:

```sh
npm run test
```

Run tests in watch mode:

```sh
npm run test:watch
```

## Project Notes

- Path alias `@` points to `src`.
- Main app entry is `src/main.tsx`.
